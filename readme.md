# ⌨️ Type Pulse

A terminal-based typing speed tester built in C++ 
as a B.Tech 1st Year project at Jaypee Institute of Technology.

## 👥 Team
- Kishita Gupta
- Shreya Agarwal
- Idika Kumar
-Diya Pant

## ✨ Features
- ⌨️ Typing speed test with WPM & accuracy calculation
- 🏆 Leaderboard — sort by WPM or Accuracy
- 🎯 Practice Mode — no leaderboard pressure
- ⚔️ Multiplayer Battle — compete with a friend
- 📝 Custom Text Mode — type your own passage
- 💾 Persistent leaderboard saved to file
- 🎨 Colored terminal UI

## 🚀 How to Run

### Windows
```bash
g++ typewriter.cpp -o typepulse.exe
typepulse.exe
```

### Linux / Mac
```bash
g++ typewriter.cpp -o typepulse
./typepulse
```

## 🛠️ Tech Stack
- Language: C++
- Concepts: OOP, Inheritance, File I/O, Classes,
  Bubble Sort, String manipulation

## 🖼️ How It Works
1. Enter your name and take a typing test
2. Your WPM and accuracy are calculated instantly
3. Score saved to leaderboard automatically
4. Challenge a friend in Multiplayer Battle mode
5. View top 10 players anytime
